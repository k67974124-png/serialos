# Codex 启动指令

这些文件是每个 Epic 的启动模板，不替代 `AGENTS.md`、任务文件或产品规格。

使用方法：

1. 在仓库根目录启动 Codex；
2. 进入 Plan 模式；
3. 复制对应文件中的“规划指令”；
4. 人工接受计划后，再复制“实施指令”；
5. 每次只完成一个垂直切片；
6. 完成后运行 `$verify-release` 和 `/review`；
7. 不要连续自动推进下一个 Epic。
